# Work Photos Directory

Upload your work photos here to showcase on the website.

## Recommended Photo Names:
- `brake-repair-1.jpg` - Brake repairs
- `diagnostic-1.jpg` - Diagnostic work
- `oil-service-1.jpg` - Oil changes
- `battery-replacement-1.jpg` - Battery work
- `suspension-1.jpg` - Suspension work
- `engine-work-1.jpg` - Engine repairs
- `mobile-service-1.jpg` - Mobile setup photos
- `customer-vehicle-1.jpg` - Customer vehicles

## Photo Requirements:
- Format: JPG or PNG
- Recommended size: 1200x800px or similar landscape ratio
- Keep file sizes under 500KB for fast loading
- Use descriptive names

## Adding Photos:
1. Upload your photos to this directory
2. Name them according to the convention above
3. The website will automatically display them in the work showcase section
